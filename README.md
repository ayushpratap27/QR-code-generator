# QR Code Generator

A web app that converts text or URLs into QR codes instantly.

## Features

- Convert any text or URL to a QR code
- Instant QR code generation
- Download generated QR codes
- Clean and simple UI

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/ayushpratap27/QR-code-generator.git
   ```
2. Open `index.html` in your browser.
3. Enter text or a URL and generate your QR code!
